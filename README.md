# Мини-RPG на Python

## О проекте / Über das Projekt
Это простой текстовый RPG проект на Python.  
Der ist ein einfaches textbasiertes RPG-Projekt in Python.

Вы можете:  
- создать и изменить профиль игрока  
- просмотреть свой профиль  
- сражаться с врагами, зарабатывая золото и опыт  
- каждый бой восстанавливает ваше здоровье до максимума  

Sie können:  
- ein Spielerprofil erstellen und bearbeiten  
- Ihr Profil ansehen  
- gegen Gegner kämpfen und Gold sowie Erfahrung sammeln  
- nach jedem Kampf wird Ihre Gesundheit auf Maximum wiederhergestellt
